# 🧠 MindPulse AI

Machine Learning Based Burnout Risk Prediction System

## 📌 Overview

MindPulse AI predicts burnout risk using lifestyle and work-related factors such as sleep, stress, exercise, screen time, work hours, and water intake.

The project uses a Random Forest Classifier trained on synthetically generated data to estimate burnout levels and provide personalized wellness recommendations.

---

## 🚀 Features

- Burnout Risk Prediction
- Random Forest Classification
- 5-Fold Cross Validation
- Feature Importance Analysis
- Correlation Heatmap
- Confusion Matrix
- Personalized Recommendations

---
